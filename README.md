# 🌍 Travel Booking Website UI

This is my first UI/UX design project created using Figma.

## 📌 Project Overview
A modern travel and tourism website interface focused on clean layout, booking flow structure, and user-friendly navigation.

## ✨ Features
- Hero section with strong CTA
- Step-based booking system
- International destinations (Turkey, Bali, Thailand)
- Punjab exploration section
- Google rating & statistics display
- Login & Sign Up navigation
- Structured footer design

## 📊 Project Stats (UI Representation)
- 50+ Destinations
- 200k+ Tourists
- 4.7 Google Rating
- 180+ Tour Captains

## 🛠 Tools Used
- Figma
- Auto Layout
- Components
- Grid system
